# start_install.sh
Скрипт предназначен для базовой настройки Ubuntu 16.04 + установка основных утелит
Скрипт написан для ОС Ubuntu Server <br>
Скрипт выполняет следующие действия: <br>
- Обновляет локальный индекс пакетов до последних изменений в репозиториях;<br>
- Обновляет систему;<br>
- Устанавливает программы (mc nano curl wget tree zip htop aptitude);<br>
- Русифицирует ОС;<br>
- Предоставляет возможность добавить нового пользователя с возможностью предоставления прав суперпользователя sudo;<br>
- Устанавливает fail2ban;<br>
- Позволяет настроить sshd_config для защиты системы.<br>
